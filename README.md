# 😀 Emoji Search App

A clean and responsive Emoji Search application built using **HTML, CSS, and JavaScript**.

https://alokdevops43.github.io/emoji-search-app/

## ✨ Features

- 🔍 Search emojis by name
- 📂 Filter emojis by category
- 📋 One-click copy to clipboard
- 🌙 Light / Dark mode
- 📱 Fully responsive
- ⚡ Fast and lightweight
- 💾 Theme saved using Local Storage

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome
- Google Fonts
